# VeloRead 产品需求文档 (PRD)

## 1. 产品概述
**VeloRead** 是一款专注于提升阅读速度与词汇积累的高级浏览器端 EPUB 阅读器。
**核心理念**：像 Apple Books 一样优雅，像 Kindle/Readwise 一样智能，像专业速读工具一样高效。

## 2. 核心功能列表

### 2.1 核心阅读体验 (Core Reader)
- **文件支持**：
    - 支持用户拖拽上传 `.epub` 格式电子书。
    - 自动解析书籍，保留目录结构、章节内容。
- **持久化存储**：
    - 使用本地数据库 (IndexedDB) 存储已导入的书籍，刷新页面不丢失。
- **进度管理**：
    - 自动保存阅读进度（章节、页码）。

### 2.2 视觉与交互 (UI/UX)
- **设计风格**：
    - 严格复刻 Apple Books 设计语言（阴影、卡片圆角、模糊背景）。
    - 极致的排版（字间距、行高、对齐）。
- **主题系统**：
    - 支持 **明亮 (Light)**、**暗黑 (Dark)**、**羊皮纸 (Sepia)** 等模式。
    - 主题切换需平滑过渡，无闪烁。
- **翻页效果**：
    - 提供平滑的仿真翻页或流畅的滑动翻页。

### 2.3 智能生词本 (Smart Vocabulary)
- **生词发现**：
    - 系统内置“常用 3000/5000 词表”。
    - 自动识别文中超出常用词表的“高级词汇”。
- **即时注解 (Instant Annotation)**：
    - 在高级词汇上方/附近展示简短的英文释义（类似 Kindle Word Wise）。
- **动态词汇库 (Dynamic Vocabulary) [新增]**：
    - **Mark as Known (标为认识)**：
        - 用户点击已标注的生词，该词被加入“熟词表”，注解消失。
        - 该操作全局生效（后续遇到同词不再标注）。
    - **Mark as Learning (标为生词)**：
        - 用户点击/长按未标注的普通词，选择“学习”，该词被加入“生词表”。
        - 系统立即为其拉取并显示注解。
    - **数据同步**：
        - 用户的熟词/生词数据在本地持久化。

### 2.4 阅读提速器 (Reading Pacer)
- **功能目标**：
    - 强制/引导视线移动，训练用户打破“默读”习惯，提升 WPM (Words Per Minute)。
- **交互方式**：
    - **Visual Guide (视觉引导)**：高亮当前阅读的句子或单词，按设定速度自动向后移动。
- **控制项**：
    - 调节速度 (WPM)。
    - 播放/暂停控制。
    - 调节引导模式（高亮、下划线、加粗）。

## 3. 非功能需求
- **性能**：打开大体积书籍（>20MB）无明显卡顿；动态注解渲染延迟 < 100ms。
- **隐私**：所有数据（书籍、词汇表）均存储在浏览器本地，不上传服务器。
- **兼容性**：支持主流现代浏览器 (Chrome, Edge, Safari)。

## 4. 后续规划 (Roadmap)
- 跨设备云同步 (需鉴权系统)。
- 引入 LLM (Gemini/GPT) 进行长难句解析。
